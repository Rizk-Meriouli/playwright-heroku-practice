# Playwright Heroku Practice

A structured **QA Automation learning project** built with **Playwright and TypeScript**, focused on practicing real-world web automation against  
https://the-internet.herokuapp.com

This repository documents **daily learning progress**, architectural decisions, and automation best practices from a QA Engineer perspective.

---

## Objectives

- Learn Playwright from a **QA Automation** standpoint (not application development)
- Build a clean, scalable automation architecture
- Practice test design using real, intentionally vulnerable web pages
- Maintain professional, readable, and reusable test code

---

## Tech Stack

- Playwright
- TypeScript
- Node.js
- Playwright Test Runner
